## API

Steam App for managing multiple steam accounts, the backend is an HTTP API, with strong Hypermedia features such as link references, embedded data and self-describing forms for data mutation. The API
is served by NodeJS, written in TypeScript.

Data persistence is provided by the PostgreSQL database engine.
